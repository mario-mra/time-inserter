## [0.0.1] - 2018-03-31
### Added
- Initial release.
- Added time inserter.
- Added property "Pre" -> String to add before the time.
- Added property "Post" -> String to add after the time.
- Added property "format" -> 24hours format selector.
- Added keybinding (Default: Ctrl+Alt+1).
